# video_win_rate
Repo for my video on why the win rate is not everything. A 90% win-rate trading strategy can still lose money.

## Viewing the dashboard

`win-rate-dashboard.html` is a self-contained interactive dashboard - no install, build step, or internet connection required.

Just open the file in any web browser:

- **Double-click** `win-rate-dashboard.html` in your file explorer, or
- Drag it into an open browser window.

Drag the sliders (or type exact values into the boxes) to change the win rate, average win/loss, fees, and trade count, and watch the expectancy, breakeven win rate, and simulated equity curve update live. Use the preset buttons for quick scenarios and **Reroll trades** to draw a fresh random run.
